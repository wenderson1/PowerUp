# Roadmap

**Current Milestone:** M4 — History & Progression
**Status:** In planning

---

## M1 — Foundation

**Goal:** Project scaffolded, local database initialized, and navigation shell working on Android.
**Status:** ✅ Complete

---

## M2 — Workout & Exercise Management

**Goal:** User can fully manage their workout plans and exercises before starting a session.
**Status:** ✅ Complete

---

## M3 — Session Execution

**Goal:** User can actively run a workout session, logging weights and marking exercises done.
**Status:** ✅ Complete

### Features

**Session Execution** — PLANNED

- "Start Workout" flow from workout detail screen
- Mark individual exercises as completed during session
- Log weight (kg) used per exercise during session
- Complete session and save to history

---

## M4 — History & Progression

**Goal:** User can review their training history and see weight progression over time.
**Target:** Both weight-per-exercise and session history are accessible and readable.

### Features

**Weight History** — PLANNED

- View last N logged weights for any exercise (chronological)
- Sparkline or simple list showing weight trend

**Workout History** — PLANNED

- Dated log of all completed workout sessions
- Tap a session to see which exercises were done and what weights were used

---

## Future Considerations

- Rest timer between sets
- Built-in exercise library / suggestions
- Export data to CSV
- iOS support via Capacitor
- Charts and progression graphs
- Personal records (PR) tracking per exercise
